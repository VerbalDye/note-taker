
  # Note Taker

  ## Description
  An express.js app that tracks notes. You can save and delete notes.

  ## Installation Instructions
  1. Download the project zip.
2. Install node.js.
3. Open a command prompt to the project directory.
4. Run `npm i`
5. Run `npm start`
6. Enjoy!
